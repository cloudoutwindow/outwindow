# OpenClaw Personal Assistant System

这是一个基于OpenClaw的个人助理系统，专为金融信息处理和知识管理而设计。

## 功能特色

### 1. 自动化A股市场分析
- 每日下午3点进行详细的A股市场分析
- 每日下午3点30分生成A股市场总结报告
- 自动收集主要指数表现、板块轮动、资金流向和重要新闻

### 2. 知识库管理
- 集成Obsidian知识管理系统
- 自动记录每日市场分析到知识库
- 支持知识图谱构建和链接

### 3. GitHub集成
- 所有变更自动推送到GitHub仓库
- 支持代码版本控制和协作

### 4. Discord群组管理
- 智能响应机制，仅在被提及或直接对话时激活
- 避免群组消息干扰

## 技术栈

- **OpenClaw**: 核心框架
- **GitHub Codespaces**: 开发环境
- **Discord**: 通信平台
- **Obsidian**: 知识管理
- **GitHub**: 版本控制

## 定时任务

- `15:00` 每日A股详细分析
- `15:30` 每日A股总结报告

## 集成技能

- `coding-agent`: 处理编程任务
- `github`: GitHub仓库管理
- `web_search`: 获取实时数据
- `obsidian`: 知识库管理
- `summarize`: 内容摘要

## 部署位置

当前运行在GitHub Codespaces环境，仓库地址: https://github.com/cloudoutwindow/outwindow

## 使用方法

直接在Discord中提及机器人或发起对话即可使用各种功能。
